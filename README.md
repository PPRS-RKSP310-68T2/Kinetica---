# Kinetica
เว็บไซต์นี้เป็นเครื่องมือช่วยคำนวณโจทย์ฟิสิกส์พื้นฐานสำหรับนักเรียน ผู้ใช้สามารถเลือกหัวข้อและกรอกค่าตัวแปรเพื่อให้ระบบคำนวณผลลัพธ์พร้อมหน่วยอย่างถูกต้อง ภายในเว็บไซต์มีหัวข้อสำคัญ ได้แก่ โพรเจกไทล์ โมเมนตัม การเคลื่อนที่แบบวงกลม กฎของโอห์ม และทัศนศาสตร์ พร้อมกราฟแสดงการเคลื่อนที่ของโพรเจกไทล์ เพื่อช่วยให้เข้าใจหลักการทางฟิสิกส์ได้ง่ายขึ้น
